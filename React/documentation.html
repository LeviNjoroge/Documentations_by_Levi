React Documentation

This is my version of a documentation for React.

It is a work in progress.

Prerequisites:

      Knowledge of the following:

      • HTML
      • CSS
      • JavaScript

      Download the following:

      • Node.js
      • NPM
      • VS Code

You should follow the following steps to get started with React:

  1. open a terminal and navigate to the directory where you want to create your React project.
  2. run the command: 'npm create vite@latest'
  3. from the list of frameworks, select 'React'
  4. select 'JavaScript' as your language
  5. after its done, run the command: 'npm install'
  6. run the command: 'npm run dev'
  7. open your browser and navigate to the specified port number

That's it! You should now have a React project up and running.

Remember to always start your project with 'npm run dev'. that is, whenever your project restarts, you can use this command to restart your project.
